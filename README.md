## 📌 프로젝트 개요

> **Voice Magician**  
> *이 프로젝트는 액션 RPG 게임으로, Unreal Engine 5를 사용하여 개발되었습니다.*  
> *프로그래머 4명이 협업하여 각자 캐릭터, 몬스터, NPC, 인벤토리 시스템을 담당했습니다.*  
> *맵 구성은 일반 필드와 보스 맵으로 나누어 설계되었습니다.*

## 🎯 게임 방법
| 조작 | 동작 |
|------|------|
| **W / A / S / D** | 이동 |
| **Shift** | 대쉬 |
| **마우스 좌클릭** | 곡사형 스킬 |
| **마우스 우클릭** | 기본 공격 |
| **Q** | 궁극기 |
| **E** | 상호작용 |
